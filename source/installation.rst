Installation
============

1. **Create a VM and install OpenSAS and its Dashboard.**

   - **OpenSAS Installation:**

     - `https://github.com/asheeshtripathi/OpenSAS <https://github.com/asheeshtripathi/OpenSAS>`_
     - `https://github.com/CCI-NextG-Testbed/OpenSAS <https://github.com/CCI-NextG-Testbed/OpenSAS>`_

   - **OpenSAS Dashboard Installation:**

     - `https://github.com/asheeshtripathi/OpenSAS-dashboard <https://github.com/asheeshtripathi/OpenSAS-dashboard>`_
     - `https://github.com/CCI-NextG-Testbed/OpenSAS-dashboard <https://github.com/CCI-NextG-Testbed/OpenSAS-dashboard>`_

2. **CBSD Creation**

   A CBSD is a CBRS device—a 5G base station with a CBSD client running on it.

   - In another VM, install the **srsRAN** project in ZMQ mode and test independent 5G end-to-end operation.
   - Place the CBSD folder, which can be downloaded from:

     - `https://github.com/asheeshtripathi/CBSD <https://github.com/asheeshtripathi/CBSD>`_
